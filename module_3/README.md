# Module 3 – Risk Assessment

This folder contains the Module 3 deliverables for the ALY6130 Risk Management Analytics Final Project.

## File
Group4_Module3_Risk_Assessment.xlsx

## Description
This module includes the risk assessment analysis developed for the ServiceNow–Armis case study. The Excel workbook contains the structured risk analysis, including additional risks identified through external research.

In addition to the workbook, the module also includes the written report and the Python notebook (.ipynb) used to generate the risk heatmap programmatically.

## Tabs Included
Problem Statement  
Risk Calculation Sheet  
Risk Register  
Key Risk Indicators (KRI)  
Risk Heatmap

## Purpose
These components support the evaluation of risks by analyzing likelihood and impact scores, documenting key risk indicators, and visualizing overall risk exposure through a heatmap created both in Excel and programmatically using Python.
