# Module 4 – Quantitative Risk Analysis

This folder contains the Module 4 deliverables for the ALY6130 Risk Management for Analytics Final Project.

## Files

- **ALY6130_M4_Assignment.ipynb** – Jupyter notebook with programmatic heatmap, ML risk classification, and Monte Carlo simulation
- **ALY6130_M4_Report.docx** – Written risk assessment report
- **ALY6130_M4_Risk_Register.xlsx** – Risk register with likelihood, impact, KRI, and treatment plans

## Description

This module applies quantitative risk analysis techniques to the ServiceNow–Armis case study:
- Part 1: Programmatic Risk Heatmap Analysis
- Part 2: Machine Learning Risk Classification (RandomForest)
- Part 3: Monte Carlo Simulation with probability-weighted outcomes
