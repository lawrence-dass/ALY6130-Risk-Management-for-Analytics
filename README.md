# ALY6130 – Risk Management for Analytics
## ServiceNow–Armis Risk Assessment Project
### Northeastern University | Group 4

This repository contains all coursework and assignments for ALY6130 Risk Management for Analytics, focusing on a comprehensive risk assessment project for the ServiceNow–Armis case study.

## Project Overview

This project applies risk management methodologies and analytical frameworks to assess risks associated with the ServiceNow–Armis business case. The work is organized into modules, each building upon previous analyses to develop a comprehensive risk management strategy.

## Repository Structure

```
ALY6130-Risk-Management-for-Analytics/
├── module_1/          # SWOT Analysis
├── module_2/          # Risk Identification & Risk Register
├── module_3/          # Risk Assessment
├── module_4/          # Quantitative Risk Analysis
└── README.md          # This file
```

## Modules

### Module 1: SWOT Analysis
**Focus:** Strategic analysis of ServiceNow–Armis through SWOT framework

**Deliverables:**
- SWOT Analysis document identifying Strengths, Weaknesses, Opportunities, and Threats
- Initial and final drafts of the analysis

**Location:** `module_1/assignment/`

---

### Module 2: Risk Identification & Risk Register
**Focus:** Systematic identification and documentation of risks

**Deliverables:**
- Risk Identification document
- Risk Register (Excel workbook) with comprehensive risk catalog
- Initial and final assignment versions

**Location:** `module_2/assignment/`

---

### Module 3: Risk Assessment
**Focus:** Quantitative and qualitative risk assessment with visualization

**Deliverables:**
- Risk Assessment Excel workbook containing:
  - Problem Statement
  - Risk Calculation Sheet
  - Risk Register
  - Key Risk Indicators (KRI)
  - Risk Heatmap
- Written risk assessment report (PDF)
- Python Jupyter notebook for programmatic heatmap generation

**Location:** `module_3/assignment/`

---

### Module 4: Quantitative Risk Analysis
**Focus:** I&W Framework, Monte Carlo simulation, and ML-based risk classification

**Deliverables:**
- Jupyter notebook with programmatic heatmap, RandomForest risk classification, and Monte Carlo simulation
- Written risk assessment report (docx)
- Risk Register Excel workbook with KRI, treatment plans, and escalation triggers
- Supporting data (feature importance, ML training data, Monte Carlo summary, risk parameters)

**Location:** `module_4/assignment/`

---
